Soal 1: Identifikasi Proposisi

Manakah dari kalimat berikut yang merupakan proposisi?

  1. 2 + 2 = 5
  2. x + 3 = 10
  3. Apakah hari ini hujan?
  4. Semoga Anda sehat selalu
  5. Java adalah bahasa pemrograman

Jawaban:

  1. Proposisi - Dapat dinilai benar/salah (dalam hal ini salah)
  2. Bukan proposisi - Kalimat terbuka, tergantung nilai x
  3. Bukan proposisi - Pertanyaan, tidak bisa dinilai benar/salah
  4. Bukan proposisi - Harapan, tidak bisa dinilai benar/salah
  5. Proposisi - Dapat dinilai benar (memang benar)

Soal 2: Nilai Kebenaran

Diketahui: P = True, Q = False, R = True. Tentukan nilai kebenaran dari:

  1. P ∧ Q
  2. P ∨ Q
  3. ¬P
  4. P → Q
  5. (P ∧ R) ∨ Q

Jawaban:

  1. P ∧ Q = True ∧ False = False
  2. P ∨ Q = True ∨ False = True
  3. ¬P = ¬True = False
  4. P → Q = True → False = False (hanya salah jika P benar dan Q salah)
  5. (P ∧ R) ∨ Q = (True ∧ True) ∨ False = True ∨ False = True

Soal 3: Tabel Kebenaran

Buat tabel kebenaran untuk ekspresi: ¬(P ∧ Q) ↔ (¬P ∨ ¬Q)

Jawaban:

P Q P ∧ Q ¬(P ∧ Q) ¬P ¬Q ¬P ∨ ¬Q ¬(P ∧ Q) ↔ (¬P ∨ ¬Q)
T T T F F F F T
T F F T F T T T
F T F T T F T T
F F F T T T T T

Kesimpulan: Ekspresi ini adalah tautologi (selalu bernilai benar). Ini adalah salah satu hukum De Morgan.

Soal 4: Penerapan dalam Kode

Diberikan variabel boolean: isMorning = true, isWeekend = false, hasCoffee = true. Tentukan nilai dari:

  1. isMorning && !isWeekend
  2. (isMorning || isWeekend) && hasCoffee
  3. !isMorning || (isWeekend && hasCoffee)

Jawaban:

  1. isMorning && !isWeekend = true && !false = true && true = true
  2. (isMorning || isWeekend) && hasCoffee = (true || false) && true = true && true = true
  3. !isMorning || (isWeekend && hasCoffee) = !true || (false && true) = false || false = false

Soal 5: Analisis Logika

Jika diketahui pernyataan "Jika hari hujan, maka jalanan basah" bernilai benar, apa yang dapat disimpulkan jika:

  1. Hari hujan
  2. Jalanan tidak basah
  3. Hari tidak hujan
  4. Jalanan basah

Jawaban: Misalkan P = "Hari hujan", Q = "Jalanan basah"

Diketahui: P → Q = True

  1. Jika P = True (hari hujan), maka Q harus = True (jalanan basah)
  2. Jika Q = False (jalanan tidak basah), maka P harus = False (hari tidak hujan)
  3. Jika P = False (hari tidak hujan), maka Q bisa True atau False (jalanan bisa basah atau tidak)
  4. Jika Q = True (jalanan basah), maka P bisa True atau False (bisa karena hujan atau sebab lain)

Catatan: Implikasi P → Q hanya salah jika P benar dan Q salah.

Soal 6: Hukum Logika

Buktikan dengan tabel kebenaran bahwa: P → Q ≡ ¬P ∨ Q

Jawaban:

P Q P → Q ¬P ¬P ∨ Q P → Q ≡ ¬P ∨ Q
T T T F T T
T F F F F T
F T T T T T
F F T T T T

Karena kolom "P → Q" dan "¬P ∨ Q" memiliki nilai yang sama untuk semua kombinasi P dan Q, maka terbukti bahwa P → Q ≡ ¬P ∨ Q.

Soal 7: Penerapan Database Query

Tulis query SQL untuk mencari data mahasiswa dengan kondisi:

  • Jurusan = "Informatika" ATAU "Sistem Informasi"
  • IPK ≥ 3.0
  • Tahun masuk = 2020 ATAU 2021
  • Bukan mahasiswa yang sedang cuti

Jawaban:

SELECT * FROM mahasiswa WHERE (jurusan = 'Informatika' OR jurusan = 'Sistem Informasi') AND ipk >= 3.0 AND (tahun_masuk = 2020 OR tahun_masuk = 2021) AND NOT status_cuti = true;

Penjelasan:

  • OR untuk memilih antara dua jurusan
  • AND untuk menggabungkan semua kondisi
  • NOT untuk mengecualikan mahasiswa yang cuti

Soal 8: Sirkuit Logika

Sebuah alarm kebakaran akan berbunyi jika:

  • Sensor asap mendeteksi asap, ATAU
  • Sensor suhu mendeteksi suhu > 50°C DAN tombol manual ditekan

Tulis ekspresi logika untuk kondisi alarm berbunyi dan buat tabel kebenarannya!

Jawaban:

Ekspresi logika: Alarm = A ∨ (S ∧ T)
Dimana:
A = Sensor asap mendeteksi asap
S = Sensor suhu > 50°C
T = Tombol manual ditekan

Tabel Kebenaran:

A S T S ∧ T A ∨ (S ∧ T)
FFFFF
FFTFF
FTFFF
FTTTT
TFFFT
TFTFT
TTFFT
TTTTT

Alarm akan berbunyi dalam 5 dari 8 kemungkinan kondisi.

Tips Mengerjakan Soal Logika:

  1. Pahami definisi operator - Ingat tabel kebenaran untuk setiap operator
  2. Gunakan tabel kebenaran untuk ekspresi kompleks
  3. Perhatikan prioritas operator - Negasi pertama, lalu AND, lalu OR
  4. Untuk implikasi P → Q, ingat bahwa ini hanya salah jika P benar dan Q salah
  5. Latihan teratur - Logika butuh pembiasaan berpikir
