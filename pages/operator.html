Operator Logika

Mengenal operator-operator dasar dalam logika boolean

Apa itu Operator Logika?

Operator logika digunakan untuk membentuk proposisi majemuk dari satu atau lebih proposisi sederhana. Operator ini mengkombinasikan nilai kebenaran dari proposisi-proposisi untuk menghasilkan nilai kebenaran baru.

Operator Logika Dasar

1. Negasi (NOT) - ¬ atau ~

Fungsi: Membalik nilai kebenaran suatu proposisi

Contoh: Jika P = "Hari ini hujan" (True), maka ¬P = "Hari ini tidak hujan" (False)

Dalam pemrograman: ! (JavaScript), not (Python)

2. Konjungsi (AND) - ∧

Fungsi: Menghasilkan nilai benar hanya jika kedua proposisi benar

Contoh: P = "Hari ini hujan" (True), Q = "Saya punya payung" (True), maka P ∧ Q = True

Dalam pemrograman: && (JavaScript), and (Python)

3. Disjungsi (OR) - ∨

Fungsi: Menghasilkan nilai benar jika setidaknya satu proposisi benar

Contoh: P = "Hari ini hujan" (True), Q = "Saya punya payung" (False), maka P ∨ Q = True

Dalam pemrograman: || (JavaScript), or (Python)

4. Implikasi (Jika-Maka) - →

Fungsi: Menghubungkan dua proposisi dalam bentuk "jika P maka Q"

Contoh: P = "Hari ini hujan" (True), Q = "Jalanan basah" (True), maka P → Q = True

Catatan: P → Q bernilai salah hanya jika P benar dan Q salah

5. Biimplikasi (Jika dan Hanya Jika) - ↔

Fungsi: Menghasilkan nilai benar jika kedua proposisi memiliki nilai kebenaran yang sama

Contoh: P = "2 + 2 = 4" (True), Q = "4 - 2 = 2" (True), maka P ↔ Q = True

Tabel Operator Logika

Operator Simbol Nama Lain Contoh Hasil
Negasi ¬, ~, ! NOT, Bukan ¬P Kebalikan dari P
Konjungsi ∧, &, · AND, Dan P ∧ Q True jika P dan Q True
Disjungsi ∨, |, + OR, Atau P ∨ Q True jika P atau Q True
Implikasi →, ⇒, ⊃ Jika-Maka P → Q False hanya jika P True dan Q False
Biimplikasi ↔, ⇔, ≡ Jika dan hanya jika P ↔ Q True jika P dan Q sama

Prioritas Operator

Urutan evaluasi operator logika (dari prioritas tertinggi ke terendah):

  1. Negasi (¬)
  2. Konjungsi (∧)
  3. Disjungsi (∨)
  4. Implikasi (→)
  5. Biimplikasi (↔)

Contoh Evaluasi Ekspresi:

Ekspresi: ¬P ∧ Q ∨ R → S

Dievaluasi sebagai: (((¬P) ∧ Q) ∨ R) → S

Langkah-langkah:

  1. Hitung ¬P
  2. Hitung (¬P) ∧ Q
  3. Hitung ((¬P) ∧ Q) ∨ R
  4. Hitung (((¬P) ∧ Q) ∨ R) → S

Contoh Penggunaan dalam Ekspresi

Misalkan:
P = "Hari ini hujan" (True)
Q = "Saya membawa payung" (False)
R = "Saya akan basah" (True)

Ekspresi 1: P ∧ ¬Q → R
= True ∧ ¬False → True
= True ∧ True → True
= True → True
= True

Ekspresi 2: (P ∨ Q) ∧ R
= (True ∨ False) ∧ True
= True ∧ True
= True

Ekspresi 3: P ↔ Q
= True ↔ False
= False
