Tabel Kebenaran

Memahami cara membuat dan membaca tabel kebenaran untuk ekspresi logika

Apa itu Tabel Kebenaran?

Tabel kebenaran adalah tabel yang menunjukkan nilai kebenaran dari suatu proposisi majemuk untuk setiap kombinasi nilai kebenaran dari proposisi penyusunnya. Tabel ini membantu kita menganalisis perilaku ekspresi logika.

Tabel Kebenaran Operator Dasar

Tabel Kebenaran untuk Satu Variabel

P ¬P
True False
False True

Tabel Kebenaran untuk Dua Variabel

P Q P ∧ Q P ∨ Q P → Q P ↔ Q
True True True True True True
True False False True False False
False True False True True False
False False False False True True

Cara Membuat Tabel Kebenaran

Langkah-langkah membuat tabel kebenaran:

  1. Tentukan jumlah variabel proposisi (n)
  2. Tentukan jumlah baris = 2n
  3. Tulis semua kombinasi nilai kebenaran untuk variabel
  4. Tambahkan kolom untuk setiap sub-ekspresi
  5. Hitung nilai kebenaran untuk setiap kolom

Contoh: Membuat Tabel Kebenaran untuk ¬(P ∧ Q)

Langkah 1: 2 variabel (P dan Q)

Langkah 2: Jumlah baris = 22 = 4

Langkah 3-5: Buat tabel:

P Q P ∧ Q ¬(P ∧ Q)
True True True False
True False False True
False True False True
False False False True

Konsep Penting dalam Tabel Kebenaran

1. Tautologi

Ekspresi logika yang selalu bernilai benar untuk semua kombinasi nilai kebenaran.

Contoh: P ∨ ¬P (Hukum excluded middle)

2. Kontradiksi

Ekspresi logika yang selalu bernilai salah untuk semua kombinasi nilai kebenaran.

Contoh: P ∧ ¬P (Hukum kontradiksi)

3. Kontingensi

Ekspresi logika yang bukan tautologi maupun kontradiksi (terkadang benar, terkadang salah).

Contoh: P ∧ Q (bergantung nilai P dan Q)

Hukum-Hukum Logika dalam Tabel Kebenaran

Hukum Contoh Keterangan
Identitas P ∧ True ≡ P True adalah identitas untuk ∧
Komutatif P ∧ Q ≡ Q ∧ P Urutan tidak penting
Asosiatif (P ∧ Q) ∧ R ≡ P ∧ (Q ∧ R) Pengelompokan tidak penting
Distributif P ∧ (Q ∨ R) ≡ (P ∧ Q) ∨ (P ∧ R) Sama seperti aljabar
De Morgan ¬(P ∧ Q) ≡ ¬P ∨ ¬Q Hukum penting untuk negasi
